# 📦 skills

Cada pasta com um `SKILL.md` é uma **Agent Skill**. As skills são organizadas em **buckets** por maturidade/categoria — só os buckets estáveis aparecem no README principal e no plugin.

| Bucket | O que vive aqui |
|---|---|
| [`in-progress/`](in-progress/) | Rascunhos em desenvolvimento — hoje, a constelação **forge** em rework. Excluído do README principal e do plugin. |

*Buckets estáveis (`engineering/`, `productivity/`, etc.) entram aqui conforme as skills graduarem.*

Instruções de instalação e a documentação completa estão no [README principal](../README.md).
