# In Progress

Skills still in development — not ready to use. Expect rough edges, breaking changes, and abandoned experiments. Excluded from the top-level README and the plugin until they graduate to a stable bucket. Right now it's all the forge constellation, under rework toward one genuinely useful version — `forge-atelier` is the auto-invoked orchestrator, the rest are explicit-only components it reads and applies.

- **[forge-atelier](./forge-atelier/SKILL.md)** — Orchestrator: forges a reusable skill out of an expert's own method.
- **[forge-desmonte](./forge-desmonte/SKILL.md)** — Functional deconstruction of the method into parts, functions, and real dependencies (Movement I).
- **[forge-grill](./forge-grill/SKILL.md)** — One-question-at-a-time interrogation that pulls out the tacit method; also grills solo plans (Movement I).
- **[forge-otimizador](./forge-otimizador/SKILL.md)** — Karpathy-style self-optimization against the harness (Movement III; expensive, never fires on its own).
- **[forge-handoff](./forge-handoff/SKILL.md)** — Cross-session handoff that routes real-use failures back into the forge.
